# 🎯 Smart Attendance System - Frontend

This is the frontend for the **Smart Attendance System**, developed using **React**, **Tailwind CSS**, and **TypeScript**. It provides a role-based interface for students, staff, and administrators to manage and view attendance with real-time face recognition.

<img width="100%" alt="Landing Page" src="https://github.com/user-attachments/assets/bcff82fb-fab1-40f3-a451-56ac88e58654" />

---

## 🚀 Features

- 🔐 Secure login (Gmail & password)
- 👨‍🎓 Role-based dashboards: Student, Staff, Admin
- 📸 Mark attendance via face recognition
- 📊 Real-time attendance display
- 📱 Responsive UI (mobile-friendly)
- 🔗 Backend API integration
